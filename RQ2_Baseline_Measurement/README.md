# RQ2 Baseline Measurement

This directory contains the code and evaluation scripts used for RQ2, which studies the effectiveness and usability of existing prompt-leakage defenses.

## Included Baselines

- `prompt_engineer/`: prompt-engineering defenses and their evaluation scripts.
- `promptKeeper/`: PromptKeeper-based detection and mitigation experiments.
- `promptObfuscation/`: PromptObfuscation-based soft-prompt defense experiments.

## Note on SysVec

The SysVec baseline is not redistributed in this repository due to compliance and redistribution constraints. Readers who need to reproduce SysVec-specific results should contact the original SysVec authors or follow the official release if one is available.
