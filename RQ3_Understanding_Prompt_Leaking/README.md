## This directory contains three scripts used for answering **Research Question 3 (RQ3)** in our paper.  

## 1. `attention_4stage_analyse_mean.py`

This script performs a **four-stage attention analysis** based on **mean attention values**.


## 2. `attention_drift_distribution_stat.py`

This script computes **statistical distributions of attention drift**.


## 3. `attention_drift_plot_phenomenon.py`

This script visualizes **attention drift phenomena**.
